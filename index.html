My Projects

Chatbot Integration with GPT

Developed a chat platform integrating OpenAI's Large Language Model (DaVinci) to answer user queries about websites and products. Optimized model prompting, segmented text input into intents, and implemented sub-models for generating relevant outputs.

Clothing Classification and Attribute Extraction

Implemented a YOLO-V8 model for detecting and classifying clothes in images. Designed a pipeline for background removal, color extraction, and comprehensive details about the detected clothes.

Bollworm Detection, Classification, and Counting

Worked with an NGO to develop a YOLO-V7 model for bollworm detection and classification in images. Implemented a pipeline for labeling, training, and providing information about required pesticides for crop protection.

Stock Market Price Prediction

Developed an application for predicting NSE stock prices using LSTM models. Retrieved live and historical data, created training datasets, and implemented a user interface for user-friendly predictions.
